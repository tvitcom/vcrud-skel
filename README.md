CRUD scaffolding for vlang
============

- [v] Index and table view of records (empty - create, on item - r, u, d)
- [v] Create (form)
- [v] Read (Item)
- [v] Update (Form)
- [v] Delete (button)

Howto
===

1. Edit file model.json according your table
2. Exec command: v run make.v
3. Run: v run main.v
